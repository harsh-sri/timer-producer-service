
## Description

Timer Producer Service to create and fetch timers

## Prerequisite
- Nodejs version: `v18.13.0`
- NestJs: `v10`
- Typescript
- Docker

## Tech Stack
- Node.js
- Typescript
- Nestjs
- MongoDB
- TypeOrm
- Kafka
- Docker
- Redis
